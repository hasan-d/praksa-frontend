# Praksa Frontend

Frontend projekat rađen u sklopu prakse.

## Sadržaj

- **HTML/CSS/JS** — statička web stranica sa Bootstrapom, dinamičkim karticama, search/filter, Fetch API
- **React app** — React aplikacija napravljena uz Vite, sa komponentama, state managementom, search/filterom i API pozivom

## Tehnologije

- HTML5, CSS3, JavaScript
- Bootstrap 5
- React 19 + Vite

## Pokretanje

### Statička stranica
Otvoriti `index.html` u browseru.

### React aplikacija
```bash
cd react-app
npm install
npm run dev
```
